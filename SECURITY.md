# SECURITY POLICY

## Supported Versions

Only the latest version of the Windows Super Smooth scripts is supported. Please ensure you are using the most recent release from the [Releases](https://github.com/theyonecodes/Windows-Super-Smooth/releases) page.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Because these scripts modify low-level Windows Kernel parameters, security is our top priority. If you discover a security vulnerability or an edge case that could lead to system instability, please do not open a public issue.

Instead, please report it via the following method:

*   **Email**: [Your Contact Email or GitHub Profile Link]
*   **Direct Message**: Contact the repository owner directly.

We will investigate the report and release a fix as soon as possible. Public disclosure should only happen after a fix has been provided to protect the community.
