## What's going on here

I write tools. If a tool annoys me, I build a better one. Simple.

- System tools for Linux and Windows — the stuff that makes your machine fast, not slow
- Security stuff that doesn't phone home — runs offline, scans your system, tells you what's broken
- Dev tools that don't ask you to sign up for anything

No AI agentic blockchain edge quantum synergy. Just code that does the job.
